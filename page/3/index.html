显示图片当然是最基础和重要的轮子，我们来看下Flutter的Image对应于安卓的Imageview是有多么的好用，封装的多实在的库。。

– Image：通过ImageProvider来加载图片

– Image.asset：用来加载本地资源图片

– Image.file：用来加载本地（File文件）图片

– Image.network：用来加载网络图片

– Image.memory：用来加载Uint8List资源（byte数组）图片

Flutter入门系列4---基础列表控件Row和Column

Posted on 2018-01-28   |   In Flutter   |  

上一篇说了Container，流水的讲了Row和Column。这里来进一步的说下.
做过web的估计很好理解这个，因为这个和Web中Flex布局很像，但坑的是不带滚动属性，不像ScrollView。如果超出了一行，在debug下面则会显示溢出的提示。

这个顶部有三个控件，红黄蓝，通过设置flex来设置了占比。

 class MyAppBar extends StatelessWidget {
  MyAppBar({this.title});

  final Widget title;

  @override
  Widget build(BuildContext context) {
    return Container(
        height: 56.0, // in logical pixels

        decoration: BoxDecoration(color: Colors.blue[500]),
        // Row is a horizontal, linear layout.
        child: Row(
          children: <Widget>[
            Expanded(
              child: Container(
                color: Colors.red,
                padding: EdgeInsets.all(5.0),
              ),
              flex: 1,
            ),
            Expanded(
              child: Container(
                color: Colors.yellow,
                padding: EdgeInsets.all(5.0),
              ),
              flex: 2,
            ),
            Expanded(
              child: Container(
                color: Colors.blue,
                padding: EdgeInsets.all(5.0),
              ),
              flex: 1,
            ),
          ],
        )
    );
  }
}
